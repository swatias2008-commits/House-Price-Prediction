# House-Price-Prediction
House Price Prediction using Linear Regression with Market Price Analysis in Python

This project predicts house prices using a Linear Regression model. The goal is to estimate the price of a house based on important features such as house size, number of bedrooms, house age, and distance from the city. The project also compares the predicted price with the market average and classifies the house into different price categories.

# Objectives
Predict house prices using Linear Regression.
Understand the relationship between house features and price.
Compare predicted prices with the market average.
Classify houses into different price categories.

# Technologies Used
Python
Pandas
NumPy
Matplotlib
Scikit-learn
